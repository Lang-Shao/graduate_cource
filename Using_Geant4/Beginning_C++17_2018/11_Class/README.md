Usage
-----

1. Using G++ compiler
------------------

> g++ Box.cpp -o Box

> ./Box

2. Using Make
--------------

> make Box

(Typing just Box, and as .cpp is a standard file suffix for C++ sources, it will find Box.cpp and call the system's default C++ compilers.)

> ./Box

3. Using CMake
---------------

> cmake .

> make

> ./Box
